Request Header Parser Microservice

Welcome to the request header parser microservice!

This service allows you to know information about your computer like ip address, language configuration of your web browser and some other information.

Take a look to the information showed below to make your request.


Example Usage:

[project url]/api/whoami

Example Output:

{"ipaddress":"::ffff:159.20.14.100","language":"en-US,en;q=0.5",
"software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
